Описание

Нужно написать Android/iOS приложение для того, чтобы сканировать QR код товара и оплачивать покупку с телефона (сначала товары добавляются в корзину, а потом их можно оплатить). Сервер должен получать пользователя, номер товара и сумму покупки, после этого следуют изменения в базе и сервер возвращает обратно ответ об успешной оплате. Windows приложением будет админка, с помощью которой можно будет добавлять товары в базу.

Задание

Желательно сделать систему аккаунтов:
- Регистрация
- Авторизация (и последующее хранение сессии)
- Счёт пользователя (пока искусственный)
Сервер должен хранить в себе:
- Список товаров (столбцы: id, name, price, code, count).
Count - это не количество оставшегося товара, а наоборот количество купленного. Code - код товара, который зашифрован в штрих коде.
- Пользователей (столбцы: id, name, login, password, cash, spend_cash). spend_cash - количество потраченных денег.

Как работает приложение

1. С помощью админки добавляются товары, цены и прочее.
2. Пользователь регистрируется
3. Пользователь авторизируется (сервер возвращает id, name, cash)
4. Пользователь сканирует товар и товар добавляется в корзину (перед добалением - отправляется запрос на сервер (отсканированый код) и с сервера возвращается name и price товара.
5. Пользователь нажимает «Оплатить» и ему предлагают приложить палец или ввести пин код.
6. Если всё ок - запрос отправляется на сервер, там проходят изменения (+ проверяется, хватит ли пользователю денег на покупку) и возвращается ответ.
7. Если ответ положительный - происходит оплата.
